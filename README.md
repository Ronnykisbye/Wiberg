# Lars Wiberg – Skilte, Have & Totalservice

Dette er en GitHub Pages klar demo-hjemmeside til Lars Wiberg.

## Struktur

```text
Wiberg/
├── index.html
├── css/
│   ├── color.css
│   ├── layout.css
│   ├── component.css
│   └── animation.css
├── js/
│   ├── app.js
│   ├── chatbot.js
│   └── form.js
└── assets/
```

## Funktioner

- Moderne hero-sektion med wow-effekt
- Nordisk farvepalette med grøn, natur, kobber og mørk premium kontrast
- Services: skilte, print, græs, hæk, fliser og bortkørsel
- Billedgalleri med relevante håndværker-/havebilleder
- Kontaktformular-demo med localStorage
- Chatbot-demo med simple svar
- Mobilvenligt layout

## Upload til GitHub Pages

Upload `index.html`, `css`, `js` og `assets` direkte i repo-root.

GitHub Pages skal stå til:

- Branch: `main`
- Folder: `/ (root)`
